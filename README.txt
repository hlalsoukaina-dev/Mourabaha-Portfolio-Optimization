# Optimisation de Portefeuille pour le Financement Mourabaha

Ce projet contient le code source associé à l'article scientifique intitulé : 
**"تدبير مخاطر التمويل بالمرابحة عبر البرمجة التربيعية: مقاربة استراتيجية لتوزيع الأصول"**.

## Description
Ce travail propose une approche basée sur la programmation quadratique pour optimiser l'allocation d'actifs dans un portefeuille de financement Mourabaha. L'objectif est de maximiser le ratio de Sharpe sous des contraintes stratégiques sectorielles.

## Structure du projet
- `main.py` : Le script principal contenant le modèle d'optimisation.
- `requirements.txt` : Liste des bibliothèques nécessaires pour exécuter le code.
